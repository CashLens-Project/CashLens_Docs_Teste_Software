# Documentação do CashLens - Teste de Software

Este repositório centraliza os documentos do projeto CashLens utilizados na disciplina de Teste de Software.

## Estrutura

- `documento_visão.pdf`: documento de visão do sistema, com a contextualização do produto, objetivos, público-alvo, escopo e visão geral da solução.
- `requisitos_de_software.pdf`: documento de requisitos de software, contendo os requisitos funcionais e não funcionais.
